# REITs发行审核标准（基础设施REITs与商业REITs）

## 一、概述

本文档详细阐述基础设施REITs与商业REITs的发行审核标准，包括申报条件、审核流程、审核要点、监管要求等内容。

---

## 二、基础设施REITs发行审核标准

### 2.1 申报条件

#### （1）项目合规性要求

| 要求项 | 具体内容 | 监管依据 |
|-------|---------|---------|
| **项目立项** | 项目已取得立项批复、备案或核准文件 | 发改投资〔2021〕1306号 |
| **规划许可** | 已取得建设用地规划许可证、建设工程规划许可证 | 相关法律法规 |
| **环评批复** | 已取得环境影响评价批复 | 环境保护法 |
| **施工许可** | 已取得建筑工程施工许可证 | 建筑法 |
| **竣工验收** | 已完成竣工验收并取得验收文件 | 建筑法 |
| **权属清晰** | 项目权属清晰，不存在权属争议 | 民法典、物权法 |

#### （2）运营稳定性要求

- **运营年限**：项目运营时间原则上不低于3年
- **现金流稳定**：项目已产生持续、稳定的现金流，投资回报良好
- **历史业绩**：项目近3年未发生重大违约或重大安全事故
- **增长潜力**：项目具有较好的增长潜力

#### （3）财务指标要求

| 指标 | 要求 | 说明 |
|-----|------|------|
| **净现金流分派率** | 不低于4% | 首发项目要求 |
| **资产负债率** | 不高于85% | 特殊项目可适当放宽 |
| **EBITDA覆盖率** | 不低于1.2倍 | 评估偿债能力 |
| **经营性现金流净额** | 为正 | 连续3年 |

#### （4）资产规模要求

- **单体资产**：原则上不低于10亿元
- **资产组合**：可由多个同类型资产组成，但单个资产不低于5亿元
- **发行规模**：原则上不低于10亿元

### 2.2 审核流程

#### （1）申报阶段

1. **发起人准备**：发起人（原始权益人）准备申报材料
2. **中介机构辅导**：基金管理人、财务顾问、法律顾问、审计机构等提供专业服务
3. **发改委审核**：向省级发展改革委申报，通过后报国家发展改革委审核
4. **证监会受理**：取得发改委批复后，向证监会提交申报材料

#### （2）受理阶段

1. **材料预审**：证监会受理部门对申报材料进行形式审查
2. **补正通知**：如材料不齐，发出补正通知
3. **正式受理**：材料齐备后，正式受理并出具受理通知书

#### （3）审核阶段

1. **问询反馈**：证监会提出审核问询
2. **回复问询**：项目方和中介机构进行回复
3. **多轮问询**：根据需要进行多轮问询
4. **审核委审议**：提交发行审核委员会审议

#### （4）注册阶段

1. **注册申请**：审核委通过后，提交注册申请
2. **注册决定**：证监会在20个工作日内做出注册决定
3. **发行上市**：注册后6个月内完成发行上市

### 2.3 审核要点

#### （1）资产合规性审核

**重点审核内容**：

1. **权属完整性**
   - 土地使用权是否完整
   - 房屋所有权是否清晰
   - 是否存在抵押、质押等权利限制
   - 是否存在权属争议

2. **建设程序合规性**
   - 是否按照基本建设程序办理各项审批
   - 是否存在未批先建、边批边建等情况
   - 竣工验收是否合格

3. **环保合规性**
   - 是否取得环评批复
   - 是否符合环保要求
   - 是否存在环保处罚记录

4. **产业政策合规性**
   - 是否符合国家产业政策
   - 是否属于国家限制或禁止发展的产业
   - 是否符合区域发展规划

#### （2）运营稳定性审核

**重点审核内容**：

1. **现金流来源**
   - 现金流来源是否清晰、稳定
   - 是否存在单一客户依赖
   - 是否存在政府补贴依赖

2. **运营成本**
   - 运营成本是否合理
   - 成本结构是否稳定
   - 未来成本上涨趋势

3. **收益预测**
   - 收益预测是否合理
   - 预测依据是否充分
   - 敏感性分析是否完整

4. **运营团队**
   - 运营团队是否专业
   - 运营经验是否丰富
   - 管理制度是否健全

#### （3）财务合规性审核

**重点审核内容**：

1. **会计核算**
   - 会计核算是否符合会计准则
   - 收入确认是否合规
   - 成本核算是否准确

2. **财务报告**
   - 近3年财务报告是否经审计
   - 审计意见是否为标准无保留意见
   - 是否存在重大会计差错

3. **税务合规**
   - 是否按时足额缴纳税款
   - 是否存在税务处罚
   - 是否享受税收优惠政策

#### （4）估值合理性审核

**重点审核内容**：

1. **估值方法**
   - 估值方法是否适当
   - 收益法参数是否合理
   - 折现率选择是否适当

2. **可比案例**
   - 可比案例是否具有可比性
   - 估值倍数是否合理
   - 市场法应用是否适当

3. **敏感性分析**
   - 敏感性分析是否充分
   - 压力测试是否完整
   - 风险揭示是否充分

### 2.4 监管关注重点

1. **政府依赖度**：关注项目对政府定价、政府补贴的依赖程度
2. **运营风险**：关注运营风险，如技术更新、竞争加剧等
3. **现金流稳定性**：关注现金流的稳定性和可持续性
4. **资产独立性**：关注资产的独立性和完整性
5. **利益输送**：关注是否存在利益输送和关联交易

## 三、商业REITs发行审核标准

### 3.1 申报条件

#### （1）项目合规性要求

与基础设施REITs基本相同，但特别关注以下内容：

| 要求项 | 具体内容 | 特别说明 |
|-------|---------|---------|
| **商业用途** | 土地和房屋用途必须为商业性质 | 不得为住宅或工业用途 |
| **消防验收** | 必须通过消防验收并取得合格证 | 商业场所的消防安全至关重要 |
| **食品安全** | 如涉及餐饮，需取得食品经营许可证 | 符合食品安全法要求 |
| **广告合规**：如涉及户外广告，需取得广告发布许可 | 广告法要求 |

#### （2）运营稳定性要求

- **运营年限**：项目运营时间原则上不低于3年（比基础设施REITs要求更严格）
- **现金流稳定**：项目已产生持续、稳定的现金流，投资回报良好
- **出租率要求**：近1年出租率原则上不低于80%（商业地产特有要求）
- **主要租户占比**：单一租户租金收入占比不超过50%（降低集中度风险）

#### （3）财务指标要求

| 指标 | 要求 | 说明 |
|-----|------|------|
| **净现金流分派率** | 不低于4% | 首发项目要求 |
| **资产收益率** | 不低于3.5% | 商业地产收益率要求 |
| **资本化率** | 符合市场水平 | 通常为4%-7% |
| **NOI margin** | 不低于60% | 净营业收入率，评估运营效率 |
| **经营性现金流净额** | 为正 | 连续3年 |

#### （4）资产规模要求

- **单体资产**：原则上不低于5亿元（比基础设施REITs要求低）
- **资产组合**：可由多个同类型资产组成，但单个资产不低于3亿元
- **发行规模**：原则上不低于8亿元

### 3.2 审核流程

与基础设施REITs审核流程基本相同，但有以下差异：

1. **发改委审核**：商业REITs不需要发改委前置审核（这是与基础设施REITs的主要区别）
2. **证监会直接受理**：直接向证监会提交申报材料
3. **审核周期**：相对较短，通常为3-6个月

### 3.3 审核要点

#### （1）资产合规性审核

**重点审核内容**：

1. **商业用途合规性**
   - 土地和房屋用途必须为商业性质
   - 是否存在改变用途的情况
   - 是否符合城市规划

2. **商业运营合规性**
   - 是否取得营业执照
   - 是否取得相关行业许可证（如食品经营许可证）
   - 是否符合消防、环保等要求

3. **商业权属完整性**
   - 土地使用权是否完整
   - 房屋所有权是否清晰
   - 是否存在抵押、质押等权利限制

#### （2）运营稳定性审核（重点关注）

**重点审核内容**：

1. **出租率分析**
   - 历史出租率变化趋势
   - 与周边可比项目对比
   - 出租率波动原因分析

2. **租户结构分析**
   - 主要租户情况
   - 租户行业分布
   - 租户集中度风险

3. **租金收入分析**
   - 租金水平与市场对比
   - 租金增长趋势
   - 租金收缴率

4. **运营成本分析**
   - 运营成本结构
   - 成本控制能力
   - 与市场对比

5. **市场竞争分析**
   - 周边竞争项目
   - 项目竞争优势
   - 市场份额分析

6. **消费市场分析**
   - 区域消费能力
   - 消费人群画像
   - 消费趋势分析

#### （3）财务合规性审核

**重点审核内容**：

与基础设施REITs基本相同，但特别关注：

1. **收入确认**
   - 租金收入确认是否符合准则
   - 扣点收入确认是否合规
   - 其他收入确认是否合理

2. **成本核算**
   - 运营成本核算是否准确
   - 推广费用核算是否合规
   - 维护费用核算是否合理

#### （4）估值合理性审核（重点关注）

**重点审核内容**：

1. **估值方法**
   - 通常采用收益法+市场法
   - 收益法参数是否合理
   - 市场法可比案例是否具有可比性

2. **资本化率选择**
   - 资本化率选择是否合理
   - 与市场水平对比
   - 风险调整是否充分

3. **收益预测**
   - 租金预测是否合理
   - 出租率预测是否保守
   - 空置率风险是否充分考虑

4. **敏感性分析**
   - 租金敏感性分析
   - 出租率敏感性分析
   - 资本化率敏感性分析

### 3.4 监管关注重点

1. **市场竞争风险**：关注周边竞争项目、市场竞争格局
2. **消费市场风险**：关注区域消费能力、消费趋势变化
3. **租户集中度风险**：关注单一租户依赖、主要租户稳定性
4. **运营管理风险**：关注运营团队、管理能力、运营效率
5. **租约到期风险**：关注租约到期时间、续约风险
6. **资产老化风险**：关注资产老化、需要更新改造的风险

## 四、两类REITs审核标准对比

| 对比维度 | 基础设施REITs | 商业REITs |
|---------|--------------|-----------|
| **发改委前置审核** | 需要 | 不需要 |
| **运营年限要求** | 原则上不低于3年 | 原则上不低于3年（更严格） |
| **出租率要求** | 无 | 近1年不低于80% |
| **单体资产规模** | 不低于10亿元 | 不低于5亿元 |
| **净现金流分派率** | 不低于4% | 不低于4% |
| **审核重点** | 资产合规性、现金流稳定性 | 运营管理、市场竞争、租户结构 |
| **审核周期** | 通常6-12个月 | 通常3-6个月 |
| **估值方法** | 收益法为主 | 收益法+市场法 |
| **主要风险** | 政策风险、运营风险 | 市场风险、竞争风险 |

## 五、发行审核常见问题

### 5.1 基础设施REITs常见问题

1. **权属不清晰**：土地或房屋权属存在瑕疵
2. **程序不合规**：存在未批先建、边批边建等情况
3. **现金流不稳定**：对政府补贴或单一客户依赖度高
4. **估值偏高**：估值参数选取过于乐观
5. **运营风险未充分揭示**：对运营风险揭示不充分

### 5.2 商业REITs常见问题

1. **出租率不达标**：历史出租率低于80%
2. **租户集中度高**：单一租户租金收入占比超过50%
3. **市场竞争激烈**：周边竞争项目多，市场份额小
4. **租金收入波动大**：租金收入不稳定，波动较大
5. **运营管理能力不足**：缺乏专业的商业运营管理团队

## 六、成功案例审核要点

### 6.1 基础设施REITs成功案例

**案例：沪杭甬高速REIT（508000）**

- **资产合规**：高速公路权属清晰，各项审批手续齐全
- **现金流稳定**：收费收入稳定，受经济周期影响小
- **估值合理**：采用收益法估值，参数选取合理
- **风险揭示充分**：充分揭示政策风险、运营风险

### 6.2 商业REITs成功案例

**案例：华润消费基础设施REIT**

- **出租率高**：历史出租率超过95%
- **租户结构合理**：单一租户租金收入占比不超过30%
- **运营能力强**：拥有专业的商业运营管理团队
- **市场位置优越**：位于核心商圈，竞争优势明显
- **估值合理**：采用收益法+市场法估值，与市场水平相当

## 八、注意事项

1. **审核标准动态调整**：监管机构会根据市场发展情况，适时调整审核标准
2. **个案差异性**：每个项目都有其特殊性，审核标准会根据项目情况灵活掌握
3. **专业判断重要性**：对于复杂项目，需要专业机构进行专业判断
4. **信息披露要求**：在审核过程中，必须充分、准确、完整地披露相关信息

---

## 九、总结

基础设施REITs与商业REITs在发行审核标准上既有共性，也有差异。共性在于都要求资产权属清晰、现金流稳定、估值合理；差异在于基础设施REITs更关注政策合规性和公益性，商业REITs更关注运营管理能力和市场竞争情况。准确理解两者的审核标准，对于项目申报和发行成功至关重要。
