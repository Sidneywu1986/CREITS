# REITs分类标准（基础设施REITs与商业REITs）

## 一、概述

根据中国证监会、国家发展改革委发布的《关于规范做好保障性租赁住房试点发行基础设施领域不动产投资信托基金（REITs）有关工作的通知》等文件，我国公募REITs产品主要分为两大类：

1. **基础设施REITs**（Infrastructure REITs）
2. **商业REITs**（Commercial REITs，又称消费基础设施REITs）

---

## 二、基础设施REITs

### 2.1 定义

基础设施REITs是指以交通基础设施、能源基础设施、市政基础设施、仓储物流基础设施、园区基础设施等基础设施项目为底层资产的不动产投资信托基金。

### 2.2 资产范围

基础设施REITs的底层资产包括但不限于以下类别：

#### （1）交通基础设施
- 高速公路
- 铁路
- 港口码头
- 机场
- 城市轨道交通

#### （2）能源基础设施
- 发电设施（光伏、风电、水电、火电等）
- 电网设施
- 天然气输送设施
- 充换电基础设施

#### （3）市政基础设施
- 污水处理厂
- 垃圾处理厂
- 供水设施
- 热力供应设施

#### （4）仓储物流基础设施
- 物流园区
- 配送中心
- 冷链仓储设施
- 智能仓储设施

#### （5）园区基础设施
- 产业园区
- 科技园区
- 创新园区
- 经济技术开发区

#### （6）保障性租赁住房
- 面向新市民、青年人的保障性租赁住房项目
- 人才公寓
- 蓝领公寓

#### （7）生态环保基础设施
- 污染治理设施
- 生态修复项目
- 固废处理设施

#### （8）新型基础设施
- 数据中心
- 5G基站
- 工业互联网
- 物联网基础设施

### 2.3 核心特征

1. **公益性**：多为关系国计民生的重要基础设施
2. **政策导向**：符合国家重大战略和发展方向
3. **现金流稳定**：具有稳定的现金流产生能力
4. **资产规模**：通常资产规模较大，单体资产不低于10亿元
5. **运营期限**：基础设施项目的运营期通常较长

### 2.4 监管要求

- 符合《公开募集基础设施证券投资基金指引（试行）》要求
- 80%以上基金资产投资于基础设施资产支持证券
- 基金管理人与专项计划管理人存在实际控制关系或受同一控制人控制
- 项目需符合国家产业政策，不属于国家产业政策禁止或限制领域

---

## 三、商业REITs（消费基础设施REITs）

### 3.1 定义

商业REITs是指以消费基础设施项目为底层资产的不动产投资信托基金，又称消费基础设施REITs，是C-REITs的重要补充。

### 3.2 资产范围

商业REITs的底层资产包括但不限于以下类别：

#### （1）购物中心
- 综合购物中心
- 奥特莱斯
- 社区商业中心

#### （2）百货商场
- 大型百货商场
- 主题百货商场
- 品牌集合店

#### （3）家居建材市场
- 品牌家居卖场
- 建材市场
- 家居体验中心

#### （4）批发市场
- 综合批发市场
- 专业批发市场
- 电子商务产业园

#### （5）品牌连锁店
- 大型连锁超市
- 品牌专卖店
- 体验式消费场所

### 3.3 核心特征

1. **消费驱动**：资产价值高度依赖消费市场活力
2. **市场导向**：受市场供需关系影响明显
3. **运营敏感**：需要专业的商业运营管理
4. **增值空间**：通过运营提升可实现资产增值
5. **租金稳定**：通常采用租金+扣点模式，现金流相对稳定

### 3.4 监管要求

- 符合《关于规范做好保障性租赁住房试点发行基础设施领域不动产投资信托基金（REITs）有关工作的通知》要求
- 项目需经营3年以上，已产生持续、稳定的现金流
- 投资回报良好，并具有持续经营能力、较好增长潜力
- 发起人（原始权益人）具有较强扩募能力
- 首发项目净现金流分派率原则上不低于4%

## 四、分类对比

| 对比维度 | 基础设施REITs | 商业REITs |
|---------|--------------|-----------|
| **资产类型** | 交通、能源、市政、仓储物流、园区等 | 购物中心、百货商场、家居建材等 |
| **政策导向** | 强政策导向，符合国家战略 | 市场导向，消费驱动 |
| **现金流特征** | 收费模式相对固定，现金流稳定 | 受消费市场影响，波动性相对较大 |
| **运营要求** | 基础运维为主 | 需要专业的商业运营管理 |
| **估值方法** | 收益法为主，参考政府定价 | 收益法+市场法，参考市场可比案例 |
| **风险特征** | 政策风险、运营风险 | 市场风险、竞争风险 |
| **增值潜力** | 相对有限 | 较大，通过运营提升可实现增值 |
| **主要投资者** | 保险资金、养老金等长期资金 | 公募基金、个人投资者等 |

## 五、分类判断标准

### 5.1 基础设施REITs判断标准

满足以下条件之一的，可认定为基础设施REITs：

1. 资产类型属于《关于规范做好保障性租赁住房试点发行基础设施领域不动产投资信托基金（REITs）有关工作的通知》明确支持的资产类别
2. 项目属于国家重大战略、国家发展改革委试点项目
3. 资产具有明显的基础设施属性，提供公共服务或准公共服务
4. 项目收费具有政府定价或政府指导价性质

### 5.2 商业REITs判断标准

满足以下条件之一的，可认定为商业REITs：

1. 资产类型属于消费基础设施，包括购物中心、百货商场、家居建材市场、批发市场等
2. 项目主要收入来源于商业经营活动，租金收入占比超过80%
3. 资产运营需要专业的商业管理团队
4. 项目价值受消费市场、商业环境、地理位置等因素影响较大

## 六、监管政策依据

### 6.1 基础设施REITs相关政策

1. 《中国证监会 国家发展改革委关于推进基础设施领域不动产投资信托基金试点相关工作的通知》（证监发〔2020〕40号）
2. 《公开募集基础设施证券投资基金指引（试行）》（证监会公告〔2021〕12号）
3. 《关于规范做好保障性租赁住房试点发行基础设施领域不动产投资信托基金（REITs）有关工作的通知》（发改投资〔2022〕1014号）
4. 《国家发展改革委关于规范做好基础设施领域不动产投资信托基金项目申报工作的通知》（发改投资〔2021〕1306号）

### 6.2 商业REITs相关政策

1. 《关于规范做好保障性租赁住房试点发行基础设施领域不动产投资信托基金（REITs）有关工作的通知》（发改投资〔2022〕1014号）
2. 《国家发展改革委关于做好基础设施领域不动产投资信托基金项目申报工作的通知》（发改投资〔2020〕1306号）

## 八、注意事项

1. **分类并非绝对**：部分资产可能兼具基础设施和商业属性，需要根据主要功能和收入来源进行判断
2. **监管动态调整**：随着REITs市场发展，监管机构可能对分类标准进行调整和完善
3. **专业判断**：对于复杂项目的分类，建议咨询专业机构或监管部门的意见
4. **信息披露**：在基金招募说明书中应当明确披露底层资产的分类情况

---

## 九、总结

基础设施REITs与商业REITs是我国公募REITs市场的两大支柱类别，各有其特点和适用场景。准确理解两者的分类标准和监管要求，对于REITs项目的申报、发行、投资和存续期管理都具有重要意义。
