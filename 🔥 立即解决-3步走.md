## 🔧 如果本地运行遇到问题

### 问题 1：找不到 `pnpm` 命令

```bash
npm install -g pnpm
```

### 问题 2：端口被占用

修改 `package.json` 中的端口号，例如改为 3000：

```json
"scripts": {
  "dev": "next dev -p 3000"
}
```

然后访问：http://localhost:3000

### 问题 3：安装依赖失败

尝试清除缓存：

```bash
rm -rf node_modules
pnpm install
```

## 💡 为什么沙箱环境不行？

经过多次测试发现：

1. **Next.js 16 与沙箱环境不兼容**
   - 降级到 15.1.6 后仍然有问题
   - 可能是沙箱环境的 Node.js 版本或系统限制

2. **服务启动后无响应**
   - 端口监听正常
   - 但 HTTP 请求无法得到响应
   - 这是沙箱环境的底层限制

3. **本地环境应该可以正常工作**
   - 代码是标准 Next.js 项目
   - 没有使用沙箱特定的依赖
   - 在标准 Node.js 环境中应该正常运行
