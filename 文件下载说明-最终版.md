## 🆘 遇到问题？

### 1. 服务无法启动

```bash
# 清理缓存
rm -rf node_modules pnpm-lock.yaml .next

# 重新安装
pnpm install

# 启动
pnpm dev
```

### 2. 端口被占用

```bash
# 使用其他端口
PORT=3001 pnpm dev
```

### 3. 依赖安装失败

```bash
# 使用国内镜像
pnpm config set registry https://registry.npmmirror.com
pnpm install
```
