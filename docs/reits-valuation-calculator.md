# REITs 估值计算器使用说明

## 功能概述

在已发行REITs页面的每个REITs项目卡片上，新增了"估值计算"按钮。点击该按钮即可弹出REITs估值计算器，进行综合估值分析。

## 使用步骤

### 1. 打开已发行REITs页面

访问 `http://localhost:5000/issued-reits`

### 2. 点击估值计算按钮

在任意REITs项目卡片的底部，点击"估值计算"按钮。

### 3. 输入估值参数

估值计算器提供三个标签页：

#### 输入参数标签页

**基础参数**：
- 当前价格：自动从REITs实时数据读取（可手动修改）
- 总股本：总发行股本（单位：万股）
- 年度分红：每份年度分红金额（单位：元）
- 分红率：自动计算（年度分红/当前价格×100%）

**DCF模型参数**：
- 预期增长率：建议输入2-5%
- 折现率：建议输入7-10%
- 终值增长率：通常设为2-3%
- 预测年限：建议5-20年

**相对估值参数**：
- 同业平均市盈率（PE）：同行业平均市盈率
- 同业平均市净率（PB）：同行业平均市净率
- 同业平均分红率：同行业平均分红率
- 市场平均市盈率：整体市场平均市盈率

### 4. 计算估值

点击"开始计算"按钮，系统会自动计算以下结果：

#### 估值结果标签页

**综合估值结果**：
- 综合估值价格：DCF（40%）+ PE法（30%）+ PB法（30%）
- 当前价格：显示当前市场价格
- 隐含资本化率：年度分红/估值价格

**DCF估值**：
- DCF估值价格：基于现金流折现的估值
- DCF总现值：所有现金流现值之和
- 上行/下行空间：相对于当前价格的涨跌幅度

**相对估值**：
- PE法估值：基于同业市盈率的估值
- PB法估值：基于同业市净率的估值
- 收益率法估值：基于同业分红率的估值

**关键财务指标**：
- 净资产价值（NAV）：每股净资产价值
- 每股运营资金（FFO）：Funds From Operations
- 调整后每股运营资金（AFFO）：Adjusted Funds From Operations

#### 分析报告标签页

**估值结论**：
- 根据综合估值价格与当前价格的关系，给出投资建议：
  - 被低估（估值价格 > 当前价格110%）：建议增持
  - 合理区间（估值价格在当前价格90%-110%）：建议持有
  - 被高估（估值价格 < 当前价格90%）：建议减持

**DCF分析**：详细说明DCF模型的计算过程和结果

**相对估值分析**：说明相对估值方法的应用和结果

**风险提示**：提醒用户估值模型的局限性和风险

### 5. 重置参数

点击"重置"按钮可恢复默认参数值。

### 6. 关闭计算器

点击右上角的关闭按钮即可关闭估值计算器。

## 估值方法说明

### 1. DCF估值法（现金流折现）

DCF（Discounted Cash Flow）估值法通过预测未来现金流并将其折现到现值来计算估值。

**计算公式**：
```
DCF价格 = Σ(未来现金流 / (1 + 折现率)^n) + 终值 / (1 + 折现率)^n
```

**适用场景**：
- 现金流稳定可预测的REITs
- 具有长期稳定分红的成熟项目

**注意事项**：
- 对增长率和折现率参数敏感
- 假设条件较多，结果具有不确定性

### 2. PE估值法（市盈率）

通过同业平均市盈率来估值。

**计算公式**：
```
PE法估值 = 每股收益 × 同业平均市盈率
```

**适用场景**：
- 盈利稳定的REITs
- 同业数据完整且可比

### 3. PB估值法（市净率）

通过同业平均市净率来估值。

**计算公式**：
```
PB法估值 = 每股净资产 × 同业平均市净率
```

**适用场景**：
- 资产质量稳定的REITs
- 房地产类型项目

### 4. 收益率法估值

通过同业平均分红率来估值。

**计算公式**：
```
收益率法估值 = 年度分红 / (同业平均分红率 / 100)
```

**适用场景**：
- 分红稳定的REITs
- 关注现金流的投资者

## 参数建议

### 增长率建议

| REITs类型 | 建议增长率 | 说明 |
|---------|----------|------|
| 仓储物流 | 3-5% | 电商驱动，增长稳定 |
| 产业园 | 2-4% | 经济周期影响较大 |
| 高速公路 | 2-3% | 相对稳定，受政策影响 |
| 保障房 | 1-2% | 收益稳定，增长有限 |

### 折现率建议

| 风险等级 | 建议折现率 | 说明 |
|---------|----------|------|
| 低风险 | 7-8% | 基础设施类 |
| 中风险 | 8-9% | 产业园、仓储物流 |
| 高风险 | 9-10% | 商业地产、创新项目 |

### 同业数据获取

建议从以下渠道获取同业数据：
- 交易所官网
- 金融数据终端（Wind、Choice）
- 研究报告
- 行业协会

## 使用建议

1. **多方法对比**：不要只依赖一种估值方法，综合多种方法的结果
2. **参数敏感性测试**：调整关键参数，观察估值结果的变化
3. **结合市场环境**：考虑市场周期、政策环境等因素
4. **定期更新**：定期更新参数和估值结果
5. **风险意识**：理解估值模型的局限性，不盲目相信结果

## 常见问题

### Q: 为什么计算器中的"年度分红"默认是0？

A: 因为当前的REITs数据源中没有包含年度分红数据，需要用户手动输入或从其他渠道获取。

### Q: DCF估值结果差异很大怎么办？

A: 检查以下参数：
- 增长率是否合理
- 折现率是否过高或过低
- 预测年限是否适当

### Q: 如何获取同业数据？

A: 建议通过以下渠道：
- 交易所官网的REITs板块
- 金融数据服务提供商
- 证券研究报告
- REITs管理人公告

### Q: 估值结果与实际价格差异很大正常吗？

A: 正常。估值结果是基于假设的计算，实际价格受市场情绪、供需关系等多种因素影响。估值仅供参考。

### Q: 计算器的估值结果准确吗？

A: 计算器提供的是基于标准估值模型的计算结果，仅供参考，不构成投资建议。实际投资决策应结合更多因素综合判断。

### 数据流

1. 用户点击"估值计算"按钮
2. 传递当前REITs的名称、代码、价格到计算器组件
3. 用户输入或修改估值参数
4. 点击计算按钮，执行估值算法
5. 展示计算结果和分析报告
